GitHub 加速服务

对 Releases、Archive、Gist、Raw 等文件提供代理加速下载服务。

请输入有效的 GitHub 链接

使用说明

本服务支持对 GitHub 的 Releases、Archive、Raw、Gist 等资源进行加速下载。

  • 支持终端命令行工具，如 git clone、wget、curl 等。
  • 支持 raw.githubusercontent.com、gist.github.com、gist.githubusercontent.com 等链接加速。
  • 注意：不支持使用 SSH Key 的方式进行 git clone。
git clone https://gh.suki.icu/github.com/nginx/nginx.git

wget https://gh.suki.icu/github.com/nginx/nginx/archive/refs/heads/master.zip

curl -O https://gh.suki.icu/raw.githubusercontent.com/nginx/agent/main/README.md
